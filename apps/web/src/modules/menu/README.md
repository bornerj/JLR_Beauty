# Module: menu

Componente de navegacao publica da aplicacao.

## Escopo
- Top nav da home/publico (`PublicMenu`)
- Top nav da pagina franquias (`FranquiasMenu`)
- Bloco de acoes (carrinho, admin, login/logout, LED de banco)

## Objetivo
Centralizar tudo que pertence ao menu para facilitar manutencao e reaproveitamento em outros projetos.
